# MindLink 配色优化计划

## 问题分析
当前配色方案使用了高饱和度的渐变（紫色 + 青色）和玻璃态设计，导致 AI 感过重，不符合校园场景的温馨、亲切氛围。

## 新配色方案

### 主色调调整
- **原方案**：靛蓝 (#6366f1) + 青色 (#22d3ee)
- **新方案**：温暖橙色 (#FF6B35) + 柔和青色 (#00B4D8)

### 色彩系统
```javascript
colors: {
  primary: '#FF6B35',     // 温暖橙色 - 代表活力
  'primary-light': '#FF9F7D',
  'primary-dark': '#E55A2B',
  secondary: '#00B4D8',   // 柔和青色 - 代表信任
  'secondary-light': '#4CC9F0',
  'secondary-dark': '#0096C7',
  accent: '#FFD166',      // 阳光黄色 - 代表希望
  neutral: {
    100: '#F8FAFC',
    200: '#F1F5F9',
    300: '#E2E8F0',
    400: '#CBD5E1',
    500: '#94A3B8',
    600: '#64748B',
    700: '#475569',
    800: '#334155',
    900: '#1E293B',
  }
}
```

### 设计调整
1. **减少玻璃态**：改为更实的背景色，增强可阅读性
2. **柔和渐变**：使用低饱和度的渐变色，避免刺眼
3. **更圆润的设计**：圆角从 2xl 调整为 lg，更贴近校园风格
4. **简化阴影**：减少阴影的模糊度，使用更自然的阴影
5. **简化动画**：保留必要的过渡，去掉过于夸张的效果
